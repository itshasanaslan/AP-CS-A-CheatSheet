Unit 8: 2D Arrays

This unit covers two-dimensional arrays, their creation, manipulation, and common operations in Java programming.

Key Concepts

  • 2D Array Declaration: Creating and initializing 2D arrays
  • 2D Array Access: Reading and writing elements
  • 2D Array Traversal: Row-major and column-major traversal
  • 2D Array Algorithms: Common operations and patterns
  • 2D Array as Parameters: Passing 2D arrays to methods
  • Ragged Arrays: Understanding non-rectangular arrays
  • Matrix Operations: Basic matrix manipulations

Example Code


// Example: Working with 2D Arrays in Java
public class TwoDArrayExamples {
  public static void main(String[] args) {
    // 2D array declaration and initialization
    int[][] matrix = {
      {1, 2, 3},
      {4, 5, 6},
      {7, 8, 9}
    };
    
    // Row-major traversal
    System.out.println("Row-major traversal:");
    for (int i = 0; i < matrix.length; i++) {
      for (int j = 0; j < matrix[i].length; j++) {
        System.out.print(matrix[i][j] + " ");
      }
      System.out.println();
    }
    
    // Column-major traversal
    System.out.println("\nColumn-major traversal:");
    for (int j = 0; j < matrix[0].length; j++) {
      for (int i = 0; i < matrix.length; i++) {
        System.out.print(matrix[i][j] + " ");
      }
      System.out.println();
    }
    
    // Sum of all elements
    int sum = calculateSum(matrix);
    System.out.println("\nSum of all elements: " + sum);
    
    // Find maximum value
    int max = findMax(matrix);
    System.out.println("Maximum value: " + max);
    
    // Transpose matrix
    System.out.println("\nTransposed matrix:");
    int[][] transposed = transpose(matrix);
    printMatrix(transposed);
    
    // Ragged array example
    int[][] ragged = {
      {1, 2, 3},
      {4, 5},
      {6, 7, 8, 9}
    };
    
    System.out.println("\nRagged array:");
    printMatrix(ragged);
  }
  
  // Calculate sum of all elements
  public static int calculateSum(int[][] arr) {
    int sum = 0;
    for (int[] row : arr) {
      for (int num : row) {
        sum += num;
      }
    }
    return sum;
  }
  
  // Find maximum value
  public static int findMax(int[][] arr) {
    int max = arr[0][0];
    for (int[] row : arr) {
      for (int num : row) {
        if (num > max) {
          max = num;
        }
      }
    }
    return max;
  }
  
  // Transpose matrix
  public static int[][] transpose(int[][] arr) {
    int rows = arr.length;
    int cols = arr[0].length;
    int[][] result = new int[cols][rows];
    
    for (int i = 0; i < rows; i++) {
      for (int j = 0; j < cols; j++) {
        result[j][i] = arr[i][j];
      }
    }
    return result;
  }
  
  // Print matrix
  public static void printMatrix(int[][] arr) {
    for (int[] row : arr) {
      for (int num : row) {
        System.out.print(num + " ");
      }
      System.out.println();
    }
  }
}

Multiple Choice Practice

Question 1:

What is the output of the following code?


int[][] arr = {
    {1, 2, 3},
    {4, 5, 6},
    {7, 8, 9}
};
System.out.println(arr[1][2]);
A. 2
B. 5
C. 6
D. 8
E. 9
Explanation: The answer is C. In a 2D array, arr[1][2] accesses the element at row 1 (second row) and column 2 (third column), which is 6.

Question 2:

Which of the following is NOT a valid 2D array declaration?

A. int[][] arr = new int[3][4];
B. int[][] arr = {{1,2}, {3,4}};
C. int[][] arr = new int[3][];
D. int[][] arr = new int[][3];
E. int[][] arr = new int[3][3];
Explanation: The answer is D. When declaring a 2D array, you must specify the number of rows first. The syntax new int[][3] is invalid because it's missing the number of rows.

Question 3:

What is the time complexity of traversing a 2D array with n rows and m columns?

A. O(1)
B. O(log n)
C. O(n * m)
D. O(nÂ²)
E. O(mÂ²)
Explanation: The answer is C. Traversing a 2D array requires visiting each element once, which means n * m operations, where n is the number of rows and m is the number of columns.

Question 4:

What is the output of the following code?


int[][] arr = new int[2][3];
for (int i = 0; i < arr.length; i++) {
    for (int j = 0; j < arr[i].length; j++) {
        arr[i][j] = i + j;
    }
}
System.out.println(arr[1][2]);
A. 1
B. 2
C. 3
D. 4
E. 5
Explanation: The answer is C. The nested loops fill the array with values where each element is the sum of its row and column indices. For arr[1][2], i=1 and j=2, so 1+2=3.

Question 5:

Which of the following is true about ragged arrays?

A. They must be square
B. They must be rectangular
C. They can have rows of different lengths
D. They must be triangular
E. They must be symmetric
Explanation: The answer is C. Ragged arrays are 2D arrays where each row can have a different length. This is different from rectangular arrays where all rows must have the same length.

Tips for Success

  • Remember that 2D arrays are arrays of arrays.
  • Use nested loops for traversal.
  • Be careful with array bounds in both dimensions.
  • Consider using enhanced for loops when possible.
  • Pay attention to row-major vs column-major traversal.
